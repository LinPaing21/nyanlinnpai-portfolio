# Nyan Linn Pai — 3D Portfolio

A Soul (Pixar)-inspired 3D portfolio built with TresJS + Vue 3. Scroll through a cinematic dark stage with spotlight beams, reflective floor, and smooth camera transitions.

![Hall of me (Soul)](./hall-of-me-scene(Soul).png)

> Pair programmed with Claude Sonnet 4.6 by Anthropic.

Live → [nyanlinnpai.from-mm.dev](https://nyanlinnpai.from-mm.dev)

---

## Stack

- **TresJS + Three.js** — 3D scene & rendering
- **Vue 3 + TypeScript** — UI & logic
- **GSAP ScrollTrigger** — scroll-driven camera
- **Tailwind CSS** — styling
- **Blender** — model creation & SVG-to-3D
- **gltf-transform** — GLB optimization (22MB → 764KB)

---

## Get Started

```bash
npm install
npm run dev
```
